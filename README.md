# brainfuck
Just want to get my brain fxxked like many people did.

## Prerequisities

* cmake.
* any C++11-compatible C++ toolchain.


## How to build?

Out-of-source-directory build is recommanded.

ex.
1. mkdir build && cd build
1. cmake ../brainfuck
1. make

## What is built?

1. `runbf`: an interpreter to run `brainfuck` code.
